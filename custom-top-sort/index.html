Custom top sort

Custom top sort for Reddit is a browser extension to get the top posts on Reddit with other options than the default last hour, day, week, month and year. With the extension, it can be filtered and sorted from the last three days, 21 hours, 7 months or whatever you want! Custom top sort currently has 600+ weekly users. Due to an update on Reddit's part, this extension stopped working with their redesign and I've been unable to find a workaround. It is however fully functional when using old.reddit.com to browse.

The extension is available for Firefox, Google Chrome and Microsoft Edge. The source code is available on Github.
